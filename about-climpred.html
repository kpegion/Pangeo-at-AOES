About climpred¶

Analysis of ensemble forecast models for climate prediction

Website: https://climpred.readthedocs.io/en/stable/

What is climpred?¶

climpred is a high-level package that leverages the scientific python ecosystem to provide an interactive experience for analyzing initialized prediction systems, from file input to publication-ready visualizations.

It is an xarray wrapper and open source.

Installation¶

If you followed the Getting Started section, then you already have MetPy installed.

Otherwise, you can install with conda

conda install -c conda-forge climpred

or pip

pip install climpred

What is it good for?¶

climpred is designed for handling ensemble re-forecast data
